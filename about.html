Man Of The Day

Man of the Day is a website that will procedurally select a command to give you depending on the day. There is also a Random Man(RandMan) button that will pick a random command to give you.
The `Man` component of this website comes from the fact that all the data served by this site comes from the man(manual) command reference.
The data used by this site has been scraped from the man-pages project by Michael Kerrisk
All source code for this project is available on my github
